# Unconsolidated Memories

This folder holds raw knowledge fragments that have **not yet been distilled** into the canonical `Docs/Memories/` knowledge base.

## Purpose
Think of this as an "inbox" for things worth remembering:
- Lessons learned from debugging sessions
- Gotchas discovered during implementation
- One-off fixes that revealed systemic facts
- Notes that don't yet fit a clean memory category

## The "Dreaming" Practice
On occasion, review the files in this folder and consolidate them:
1. Extract the durable facts (what is always true going forward)
2. Write or update a file in `Docs/Memories/` with the distilled knowledge
3. Delete or archive the raw source file from this folder

**Do not treat files here as authoritative.** They are raw material, not refined knowledge.
